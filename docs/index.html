Electron将棋

Electron将棋は将棋の対局や研究を行うためのアプリです。
USIプロトコル対応のエンジンを同じPCにインストールすると対局や検討ができます。
Electron将棋はソースコードを公開しています。
誰でも開発に参加したり、派生したバージョンを作ることができます。
